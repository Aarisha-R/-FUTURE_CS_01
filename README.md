# -FUTURE_CS_01
Cyber Security Internship — Future Interns Fellowship Program | All assigned tasks and project work submitted as part of the official internship curriculum.
This report documents the results of a Vulnerability Assessment conducted on the OWASP Juice Shop application using OWASP ZAP. The assessment focused on identifying common web application security weaknesses, including insecure cookie configurations and missing HTTP security headers.
The primary objective of this assessment was to gain practical experience in web application security testing and vulnerability reporting. As this is an introductory-level vulnerability assessment project, the findings mainly consist of medium and low-risk vulnerabilities discovered through passive scanning techniques.

Disclaimer
This vulnerability assessment report represents a point-in-time evaluation of the target application. New vulnerabilities may emerge over time due to application updates, configuration changes, or newly discovered security issues.
This assessment was conducted strictly for educational purposes in a controlled environment using an intentionally vulnerable application. No unauthorized exploitation or harmful activity was performed.
